# Rich ship-page content — what to send me

Send it in any rough format (a list per ship is perfect). I turn facts into original prose,
stamp it verified, and never publish prices. Anything you skip stays an honest gap.

## PER SHIP (different for each ship)
For each ship, give me whatever you have of:

1. **Overview** — a few key facts about the ship (size, style, who it's for, standout feature).
   *(I write the original 2–3 sentence copy from these facts.)*
2. **Food & dining** — the venues, one per line, as: `Name | type | included or extra`
   - type = main dining / buffet / casual / specialty
   - e.g. `The Catch | specialty | extra`, `World Fresh Marketplace | buffet | included`
3. **Activities** — signature things to do (pools, waterpark, sports, spa, top-deck features…).
4. **Entertainment** — shows, theatres, live music, clubs, casino…
5. **Decks** — number of passenger decks (a single number).
6. **Neighbourhoods / zones** *(optional)* — named areas (e.g. "The Sanctuary", "Central Park").
7. **Bridge cam** — yes / no (does this ship have a live bridge cam on board?).
8. **Photos** — send the image files (JP/PNG) you're cleared to use commercially, with a 1-line
   caption each. **No cruise-line logos / trademarked art / copyrighted press photos.** I host them
   on our own site (fast, no third-party). If you don't have any, I'll add royalty-free
   illustrative ocean/destination images instead.

## PER LINE (same across the whole fleet — send once per line)
9. **Drink packages** — each package as: `Name | what it includes` (NO prices).
   - e.g. `Plus Beverage Package | drinks up to $15, wifi, gratuities`  →  *(I'll drop the price part)*
10. **Kids / teens / families** — nursery, kids club, teen space, and any family-cabin notes.

```
Princess (whole line)

- kids: The Youth & Teen Center; Camp Discovery (age groups)
```

That's it — one line's worth (like Princess) and I'll build every ship in that fleet, then show
you before doing the rest.
