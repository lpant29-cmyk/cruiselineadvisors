> Last updated: 2026-07-20.

## Where we are
**22 rich guides published** (all P1 + the first P2 batch + the 5 upgraded basics). Every guide is
bilingual, has a photo hero, key takeaways, graphic cards, an FAQ with schema, and contextual
interlinks. They surface automatically on the home, line, ship and destination pages.

## The daily workflow
Each day, say **"publish today's guide"** (or name one) and I will: build it rich + bilingual, run
the guard/tests, deploy it, flip its status here and in the backlog, and set the internal links. To
keep quality high we publish **one polished guide per day**, not a rushed batch.

## Rules for every daily guide (non-negotiable)
- 100% original, hand-written, factual. No prices, no invented facts, no copied prose (Seven Hard Rules).
- No em dashes. Bilingual EN + ES. Passes the banned-term guard.
- Rich template: photo hero, key takeaways, TOC, graphic cards, FAQ + Article schema.
- At least 3 natural internal links; entity-specific guides set for_lines/for_ships/for_dests so they
  reverse-link to their page. Ends in a natural call CTA.
- Tag it in `GUIDE_META` (category + contexts) so it auto-surfaces on the right pages.

## Measuring it
Watch Search Console + GA4 (via GTM) for impressions/clicks per guide, queries won, appearances in
AI overviews/snippets, and calls (`call_click`) from guide pages. Double down on the clusters that
earn traffic; refresh older guides on the 30-day fact cycle when a policy changes.
